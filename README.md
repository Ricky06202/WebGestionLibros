# Gestor de Libros

## Iniciar el Proyecto

```bash
npm i
```

```bash
npm run dev
```

## Iconos

### Pagina de Iconos

[Iconify](https://icon-sets.iconify.design)

### Componente de Iconos

#### Ejemplo de Uso

##### Astro

```html
<Icon name="solar:home-bold" />
```

##### React

```html
<Icon icon="solar:home-bold" />
```

##### Svelte

```html
<Icon icon="solar:home-bold" />
```

### Instalar conjunto de iconos

@iconify-json/\*

##### npm

```bash
npm install @iconify-json/solar
```

##### yarn

```bash
yarn add @iconify-json/solar
```
